Conceptos Básicos de CSS

En esta página vamos a aprender los fundamentos de CSS. Abajo verás ejemplos prácticos de código con sus resultados en un iFrame, para que veas cómo funcionan sin afectar esta página directamente.

1. Selectores CSS

Los selectores permiten aplicar estilos a elementos HTML. Aquí tienes algunos ejemplos básicos:

p { color: blue; } /* Cambia el color del texto de los párrafos a azul */

Ejemplo:

2. Propiedades de Texto

Las propiedades relacionadas con el texto, como el tamaño, alineación y tipo de letra, son esenciales.

h1 { font-size: 2em; text-align: center; } /* Cambia el tamaño y alinea el título */ p { font-family: 'Arial', sans-serif; } /* Define la tipografía de los párrafos */

Ejemplo:

3. Colores y Fondos

CSS permite cambiar colores de texto, fondos y bordes. Aquí algunos ejemplos:

body { background-color: #f0f0f0; } p { color: #333; background-color: #e0e0e0; padding: 10px; }

Ejemplo:

4. Bordes, Márgenes y Paddings

Los márgenes, bordes y paddings son propiedades clave para la disposición de los elementos.

div { border: 2px solid #333; padding: 10px; margin: 20px; }

Ejemplo:

5. Modelo de Caja

El modelo de caja en CSS define cómo se comportan los elementos en cuanto a ancho, alto, márgenes y relleno.

.box { width: 200px; height: 100px; padding: 20px; border: 5px solid #333; margin: 10px; }

Ejemplo:

6. Layout con Flexbox

Flexbox es una técnica muy poderosa para crear layouts flexibles y alineaciones avanzadas.

.container { display: flex; justify-content: space-between; } .item { background-color: #ddd; padding: 20px; width: 100px; }

7. Buenas Prácticas

Al escribir CSS, es importante seguir buenas prácticas como el uso de nombres de clases descriptivos, separar el estilo del contenido y evitar el uso excesivo de IDs como selectores.

HTML

Introducción

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Utiliza etiquetas para estructurar el contenido en un documento que puede ser interpretado por los navegadores.

Estructura Básica de un Documento HTML

<!DOCTYPE html>
  <html lang="es">
  <head>
      <meta charset="UTF-8">
      <title>Título de la Página</title>
  </head>
  <body>
      <!-- Contenido va aquí -->
  </body>
  </html>

Crear la estructura básica de un documento HTML5 utilizando atajos en los editores de código:

  • En Visual Studio Code o Sublime Text:
  • Escribe html:5 y presiona Tab.
  • Escribe ! y presiona Tab.

Comentarios en HTML

Los comentarios en HTML se utilizan para dejar notas o explicaciones en el código que no serán visibles en el navegador. Son útiles para documentar el código y facilitar su comprensión.

<!-- Este es un comentario en HTML -->
  <p>Este es un párrafo.</p>
  <!--
    Este es un comentario
    que abarca múltiples
    líneas.
  -->

Etiquetas Básicas de HTML

  • Encabezados (de <h1> a <h6>)
  • <p> - Define un párrafo.
  • <a> - Define un hipervínculo usando el atributo href.
  • <img> - Define una imagen, con atributos src y alt.
  • <ul> y <ol> - Listas no ordenadas y ordenadas respectivamente.
  • <div> y <span> - Definen divisiones y secciones en línea.
  • <form> - Define un formulario con campos de entrada.
  • <table> - Define una tabla con filas (<tr>) y celdas (<td> y <th>).

Atributos Globales

  • id: Define un identificador único.
  • class: Define una o varias clases.
  • style: Define estilos CSS en línea.
  • title: Proporciona un texto emergente adicional.

Etiquetas Semánticas

Las etiquetas semánticas proporcionan un significado claro sobre la estructura del contenido. Algunas de ellas son:

  • <header> - Encabezado de una página o sección.
  • <nav> - Sección de navegación.
  • <section> - Sección del documento.
  • <article> - Contenido independiente.
  • <aside> - Contenido adicional o complementario.
  • <footer> - Pie de página.

Buenas Prácticas

  1. Usa etiquetas semánticas para mejorar accesibilidad y SEO.
  2. Cierra correctamente las etiquetas.
  3. Escribe un código limpio y bien estructurado.
  4. Documenta el código con comentarios.
  5. Valida tu HTML en servicios como el W3C Markup Validation Service.

CSS

Introducción

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de documentos HTML. CSS controla el diseño y el formato de las páginas web.

Comentarios en CSS

Los comentarios en CSS se utilizan de manera similar a HTML. Se inician con /* y terminan con */. 

/* Este es un comentario en CSS */
  body {
    font-family: Arial, sans-serif;
  }

Cómo Enlazar CSS a un Documento HTML

El método recomendado es enlazar un archivo CSS externo con la etiqueta <link> en el <head> del documento HTML. 

<link rel="stylesheet" href="estilos.css">

Selectores Básicos de CSS

  • Selector de Elemento: Aplica estilos a todos los elementos de un tipo.
  • Selector de Clase: Aplica estilos a elementos con una clase específica, usando . seguido del nombre de la clase.
  • Selector de ID: Aplica estilos a un elemento con un ID específico, usando #.

Buenas Prácticas en CSS

  1. Usa archivos CSS externos para mantener el HTML limpio.
  2. Utiliza nombres de clases descriptivos.
  3. Comenta el código CSS para explicar secciones complejas.
  4. Organiza el CSS en bloques lógicos y usa unidades relativas como em o rem.
/* Buena práctica de nombres */ .boton-primario { background-color: #007BFF; color: white; padding: 10px 20px; border: none; border-radius: 5px; }
