Cuadernillo de CSS y Flexbox

Introducción a CSS

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de documentos HTML. Con CSS, puedes controlar el diseño y el formato de múltiples páginas web a la vez, lo que te permite definir cómo los elementos HTML deben aparecer en pantalla, impresos, o en otros medios.

Propiedades y valores CSS

Algunos ejemplos de propiedades CSS comunes y sus valores:

  • Color: Ajusta el color de fondo, texto, bordes, etc. 
    color: #ff0000;
  • Fuente: Define la familia, tamaño, estilo y peso de la fuente. 
    font-family: Arial; font-size: 16px;
  • Espaciado: Controla el margen, relleno y espacio entre líneas. 
    margin: 10px; padding: 5px;
  • Bordes: Define el estilo, color y ancho de los bordes. 
    border: 2px solid #000;

Modelo de Caja

El modelo de caja CSS es esencial para entender cómo los elementos son visualizados en la pantalla. Cada elemento HTML se compone de cuatro áreas:

  • Contenido: Lo que se muestra (texto, imágenes).
  • Relleno (padding): Espacio entre el contenido y el borde.
  • Borde (border): Delimita el área del contenido.
  • Margen (margin): Espacio entre el borde y otros elementos.
div { margin: 10px; padding: 5px; border: 2px solid #000; }

Introducción a Flexbox

Flexbox es un modelo de diseño en CSS que permite organizar y alinear elementos dentro de un contenedor de manera flexible, adaptándose a diferentes resoluciones y tamaños de pantalla. Es especialmente útil para crear layouts responsivos y dinámicos.

Principios de Flexbox

Flexbox se basa en dos componentes principales:

  • Contenedor Flex (flex container): El contenedor que define el contexto Flexbox.
  • Elementos Flex (flex items): Los elementos hijos dentro del contenedor que se alinean de manera flexible.

Propiedades clave de Flexbox

  • display: flex: Convierte un contenedor en un contenedor flex. 
    .container { display: flex; }
  • flex-direction: Define la dirección en la que los elementos hijos se organizan. 
    .container { flex-direction: row; }
  • justify-content: Controla la alineación a lo largo del eje principal. 
    .container { justify-content: space-between; }
  • align-items: Controla la alineación a lo largo del eje transversal. 
    .container { align-items: center; }
  • flex-grow, flex-shrink, flex-basis: Controlan el tamaño de los elementos flexibles. 
    .item { flex: 1 1 100px; }

Ejemplos de Layouts con Flexbox

Ejemplo 1: Usando flex-basis

Este ejemplo muestra cómo dividir un contenedor en dos columnas de igual tamaño usando flex-basis:

.container { display: flex; flex-wrap: wrap; } .column { flex-basis: 50%; padding: 10px; background-color: lightblue; }

Ejemplo 2: Usando width y porcentajes

Este ejemplo muestra cómo crear columnas flexibles con porcentajes:

.container { display: flex; flex-wrap: wrap; } .column { width: 50%; padding: 10px; background-color: lightgreen; }

Buenas Prácticas con Flexbox

  • Mantén la estructura semántica: Usa etiquetas semánticas como <header>, <nav>, <section>, <article>, y <footer>.
  • Uso de porcentajes: Utiliza flex-basis o width en porcentajes para que los contenedores sean más flexibles y responsivos.
  • Combina Flexbox con media queries: Usa media queries para ajustar el diseño en diferentes tamaños de pantalla.
  • Evita dimensiones fijas: Usa unidades flexibles como %, em, o rem en lugar de px.
