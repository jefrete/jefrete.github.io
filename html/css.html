Introducción a CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de documentos HTML. CSS controla el diseño y el formato de múltiples páginas web a la vez, permitiendo definir cómo los elementos HTML deben aparecer en pantalla, impresos o en otros medios.

Comentarios en CSS

Los comentarios en CSS se utilizan de manera similar a los de HTML, para agregar notas o explicaciones en el código CSS. Los comentarios comienzan con /* y terminan con */.


                  /* Este es un comentario en CSS */
                  body {
                      font-family: Arial, sans-serif;
                  }
                  /*
                  Este es un comentario
                  que abarca múltiples
                  líneas.
                  */

Cómo Enlazar CSS a un Documento HTML

Para enlazar un archivo CSS externo, utiliza la etiqueta <link> dentro del <head> de tu documento HTML:


                  <!DOCTYPE html>
                  <html lang="es">
                  <head>
                      <meta charset="UTF-8">
                      <title>Título de la Página</title>
                      <link rel="stylesheet" href="estilos.css">
                  </head>
                  <body>
                      <!-- Contenido va aquí -->
                  </body>
                  </html>

CSS Interno (Mala Práctica)

Incluir CSS directamente en el archivo HTML no es recomendable. Se puede usar la etiqueta <style> dentro del <head>:


                  <!DOCTYPE html>
                  <html lang="es">
                  <head>
                      <meta charset="UTF-8">
                      <title>Título de la Página</title>
                      <style>
                          body {
                              font-family: Arial, sans-serif;
                              background-color: #f0f0f0;
                          }
                      </style>
                  </head>
                  <body>
                      <!-- Contenido va aquí -->
                  </body>
                  </html>

CSS en Línea

Puedes aplicar estilos directamente a un elemento HTML utilizando el atributo style:


                  <p style="color: red;">Este es un párrafo con estilo en línea.</p>

Selectores Básicos de CSS

Selectores de Elemento

Aplica estilos a todos los elementos de un tipo específico.


                  p {
                      color: blue;
                  }

Selectores de Clase

Aplica estilos a elementos con una clase específica. Utiliza el punto (.) seguido del nombre de la clase.


                  
                  <p class="intro">Este es un párrafo con clase "intro".</p>
  
                  
                  .intro {
                      font-size: 20px;
                  }

Selectores de ID

Aplica estilos a un elemento con un ID específico. Utiliza la almohadilla (#) seguida del nombre del ID.


                  
                  <div id="header">Este es el encabezado.</div>
  
                  
                  #header {
                      background-color: #333;
                      color: white;
                  }

Buenas Prácticas en CSS

  1. Usa Archivos CSS Externos: Mantén tu HTML limpio y separado de la lógica de estilos.
  2. Nombres de Clases Descriptivos: Usa nombres de clases claros y significativos.
    3. 
                      .boton-primario {
                          background-color: blue;
                          color: white;
                      }
  3. Comentarios en CSS: Utiliza comentarios para explicar secciones complejas.
    4. 
                      /* Estilos para el botón principal */
                      .boton-primario {
                          background-color: blue;
                          color: white;
                      }
  4. Estructura y Organización: Organiza tus reglas CSS en bloques lógicos.
    5. 
                      /* Estilos para el cuerpo */
                      body {
                          font-family: Arial, sans-serif;
                          background-color: #f0f0f0;
                      }
  
                      /* Estilos para el encabezado */
                      header {
                          background-color: #333;
                          color: white;
                          padding: 10px;
                      }
  5. Usa Unidades Relativas: Prefiere unidades relativas (%, em, rem) sobre unidades absolutas (px) para una mejor adaptación a diferentes pantallas.
    6. 
                      body {
                          font-size: 16px;
                      }
                      .contenido {
                          font-size: 1.2em; /* 1.2 veces el tamaño de la fuente del cuerpo */
                      }

Malas Prácticas en CSS

  1. Evita el CSS en Línea: Es difícil de mantener y no reutilizable.
    2. 
                      
                      <p style="color: red;">Este es un párrafo con estilo en línea.</p>
  2. No Abuses de los Selectores ID: Son muy específicos y pueden causar problemas de mantenibilidad.
    3. 
                      /* Malo */
                      header {
                          background-color: red;
                      }
