Introducción a CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de documentos HTML. CSS controla el diseño y el formato de múltiples páginas web a la vez, permitiendo definir cómo los elementos HTML deben aparecer en pantalla, impresos o en otros medios.

Comentarios en CSS

Los comentarios en CSS se utilizan de manera similar a los de HTML, para agregar notas o explicaciones en el código CSS. Los comentarios comienzan con /* y terminan con */.


/* Este es un comentario en CSS */
body {
    font-family: Arial, sans-serif;
     }
/*
Este es un comentario que abamúltipleslíneas.
*/

Cómo Enlazar CSS a un Documento HTML

Para enlazar un archivo CSS externo, utiliza la etiqueta <link> dentro del <head> de tu documento HTML:


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la Página</title>
    <link rel="stylesheet" href="estilos.css">
</head>
<body>
    <!-- Contenido va aquí -->
</body>
</html>

CSS Interno (Mala Práctica)

Incluir CSS directamente en el archivo HTML no es recomendable. Se puede usar la etiqueta <style> dentro del <head>:


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la Página</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
             }
    </style>
</head>
<body>
    <!-- Contenido va aquí -->
</body>
</html>

CSS en Línea

Puedes aplicar estilos directamente a un elemento HTML utilizando el atributo style:


<p style="color: red;">Este es un párrafo con estilo en línea.</p>

Selectores Básicos de CSS

Selectores de Elemento

Aplica estilos a todos los elementos de un tipo específico.


                  p {
                      color: blue;
                  }

Selectores de Clase

Aplica estilos a elementos con una clase específica. Utiliza el punto (.) seguido del nombre de la clase. 



<p class="intro">Este es un párrafo con clase "intro".</p>


.intro {
        font-size: 20px;
        }

Selectores de ID

Aplica estilos a un elemento con un ID específico. Utiliza la almohadilla (#) seguida del nombre del ID. 



<div id="header">Este es el encabezado.</div>
  

#header {
    background-color: #333;
    color: white;
        }

Buenas Prácticas en CSS

  1. Usa Archivos CSS Externos: Mantén tu HTML limpio y separado de la lógica de estilos.
  2. Nombres de Clases Descriptivos: Usa nombres de clases claros y significativos.
    3. 
                      .boton-primario {
                          background-color: blue;
                          color: white;
                      }
  3. Comentarios en CSS: Utiliza comentarios para explicar secciones complejas.
    4. 
                      /* Estilos para el botón principal */
                      .boton-primario {
                          background-color: blue;
                          color: white;
                      }
  4. Estructura y Organización: Organiza tus reglas CSS en bloques lógicos.
    5. 
/* Estilos para el cuerpo */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
     }
  
/* Estilos para el encabezado */
header {
    background-color: #333;
    color: white;
    padding: 10px;
       }
  5. Usa Unidades Relativas: Prefiere unidades relativas (%, em, rem) sobre unidades absolutas (px) para una mejor adaptación a diferentes pantallas.
    6. 
body {
    font-size: 16px;
}
.contenido {
    font-size: 1.2em; /* 1.2 veces el tamaño de la fuente del cuerpo */
}

Malas Prácticas en CSS

  1. Evita el CSS en Línea: Es difícil de mantener y no reutilizable.
    2. 

<p style="color: red;">Este es un párrafo con estilo en línea.</p>
  2. No Abuses de los Selectores ID: Son muy específicos y pueden causar problemas de mantenibilidad.
    3. 
/* Malo */
header {
    background-color: red;
}

¿Qué es CSS?

CSS o Cascading Style Sheets, es una hoja de estilo que permite a los diseñadores definir el aspecto de un documento HTML. Este lenguaje de diseño es fundamental para crear páginas web atractivas y profesionales.

Sintaxis y estructura de CSS

CSS se escribe en una sintaxis sencilla con reglas que definen la apariencia de los elementos HTML. Una regla contiene un selector, que indica el elemento a estilizar, y una declaración con propiedades y valores para modificar su diseño. 


/* Sintaxis Básica */
Selector {
    Propiedad: Valor;
}
  • Selector: Identifica los elementos HTML a estilizar.
  • Propiedad: Define el aspecto del elemento (color, tamaño, margen, etc.).
  • Valor: Especifica el valor de la propiedad.

Selectores CSS

Los selectores CSS determinan a qué elementos HTML se aplican las reglas de estilo. Hay diversos tipos de selectores con diferentes niveles de especificidad.

  • 1. Selectores de Elemento: Se aplican a todos los elementos del mismo tipo.
    • 
/* Ejemplo: */
p {
   color: blue;
  }
  • 2. Selectores de Clase: Se aplican a elementos que comparten una clase específica. 
    • 
/* Ejemplo: */
.nombre-clase {
    font-weight: bold;
}
  • 3. Selectores de ID: Se aplican a un elemento único con un ID específico.
    • 
/* Ejemplo: */
#encabezado {
background-color: red;
            }
  • 4. Selectores Combinados: Se combinan para crear reglas más específicas.
    • 
    /* Ejemplo: */
    h1.destacado {
        font-size: 24px;
    }

Propiedades y valores CSS

Las propiedades CSS definen las características de los elementos HTML, como el color, el tamaño de fuente o el margen. Los valores especifican los datos concretos que se aplicarán a cada propiedad.

  • Color: Ajusta el color de fondo, texto o bordes. Ejemplo: color: #ff0000;
  • Fuente: Define la familia, el tamaño y el estilo de la fuente. Ejemplo: font-family: Arial; font-size: 16px;
  • Espaciado: Controla el margen, el relleno y el espacio entre líneas. Ejemplo: margin: 10px; padding: 5px;
  • Bordes: Define el estilo, color y ancho de los bordes. Ejemplo: border: 2px solid #000;

Modelo de caja CSS

El modelo de caja CSS es un concepto fundamental que describe cómo se organizan los elementos en el diseño web. Cada elemento HTML se visualiza como una caja con diferentes componentes que definen su espacio y apariencia.

  • Contenido: El contenido real del elemento, como texto, imágenes o videos.
  • Relleno: Espacio entre el contenido y el borde del elemento.
  • Borde: Línea que delimita el elemento, con estilo, color y ancho.
  • Margen: Espacio entre el elemento y otros elementos adyacentes.

Estilos de texto y fuentes en CSS

Las propiedades de texto y fuente en CSS permiten personalizar la apariencia del texto en las páginas web. Se puede controlar la familia de fuente, el tamaño, el estilo, el peso, el color, la alineación y otros aspectos del texto.

Propiedad Descripción
font-family Define la familia de fuente a utilizar.
font-size Establece el tamaño de la fuente.
font-weight Controla el grosor de la fuente.
color Define el color del texto.
text-align Alinea el texto dentro del elemento.
line-height Ajusta el espacio entre líneas de texto.

Flexbox en CSS

¿Qué es Flexbox?

Flexbox es un modelo de diseño en CSS que proporciona un método flexible y potente para organizar y alinear elementos dentro de un contenedor.

Principales propiedades de Flexbox

  • Alineación: Controla la alineación de los elementos dentro del contenedor, tanto vertical como horizontalmente.
  • Dirección: Determina la dirección del flujo de los elementos dentro del contenedor, como fila o columna.
  • Justificación: Distribuye el espacio entre los elementos dentro del contenedor, creando un diseño equilibrado.
  • Orden: Permite cambiar el orden de los elementos sin modificar el código HTML.

Alineación en Flexbox

Flexbox permite alinear elementos dentro de un contenedor de manera flexible. La propiedad align-items controla la alineación vertical, mientras que justify-content se encarga de la alineación horizontal. 


                .contenedor {
                    display: flex;
                    justify-content: center; /* Alineación horizontal */
                    align-items: flex-start; /* Alineación vertical */
                }

Dirección de Flexbox

Flexbox permite controlar la dirección del flujo de los elementos dentro de un contenedor. Esto se logra mediante la propiedad flex-direction.

  • row: Los elementos se disponen en una fila horizontal.
  • column: Los elementos se disponen en una columna vertical.
  • row-reverse: Los elementos se disponen en una fila horizontal inversa.
  • column-reverse: Los elementos se disponen en una columna vertical inversa.

Justificación en Flexbox

La propiedad justify-content en Flexbox permite controlar la distribución de los elementos a lo largo del eje principal del contenedor. 


                .contenedor {
                    display: flex;
                    justify-content: space-between; /* Distribuye espacio entre los elementos */
                }

Orden en Flexbox

Flexbox permite controlar el orden de los elementos dentro de un contenedor mediante la propiedad order. Asigna un valor numérico para determinar la posición de cada elemento. 


                .elemento1 {
                    order: 2; /* Se mostrará después del elemento con order 1 */
                }
                .elemento2 {
                    order: 1; /* Se mostrará primero */
                }

Buenas Prácticas de Flexbox

  • Mantén la Estructura Semántica: Usa etiquetas HTML semánticas (<header>, <section>, <footer>).
  • Uso de Porcentajes y Unidades Relativas: Utiliza porcentajes (%) y unidades como em, rem o vh/vw para tamaños flexibles y responsivos.
  • Define Proporciones con Flexbox: Usa flex-grow, flex-shrink y flex-basis para controlar el crecimiento y contracción de los elementos.
  • Minimiza la Dependencia de Dimensiones Fijas: Evita el uso excesivo de px para anchos y alturas, usa flex-basis o porcentajes.
