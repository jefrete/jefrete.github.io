HTML

Introducción

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Utiliza etiquetas para estructurar el contenido en un documento que puede ser interpretado por los navegadores.

Estructura Básica de un Documento HTML

    <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
    <title>Título de la Página</title>
    </head>
    <body>
        <!-- Contenido va aquí -->
    </body>
    </html>

Puedes crear la estructura básica de un documento HTML5 utilizando atajos en editores de código:

  • En Visual Studio Code o Sublime Text, escribe html:5 y presiona Tab.
  • Escribe ! y presiona Tab para lo mismo.

Comentarios en HTML

Los comentarios se utilizan para dejar notas que no serán visibles en el navegador:

<!-- Este es un comentario en HTML -->
<p>Este es un párrafo.</p>
<!--
Este es un comentario
que abarca múltiples
líneas.
-->

Etiquetas Básicas de HTML

Encabezados

Las etiquetas <h1> a <h6> definen encabezados de diferentes niveles: 

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>

Párrafos

La etiqueta <p> define un párrafo:

<p>Este es un párrafo.</p>

Enlaces

La etiqueta <a> define un enlace:

<a href="https://www.ejemplo.com">Visita Ejemplo</a>

Imágenes

La etiqueta <img> define una imagen. Los atributos src y alt son importantes:

<img src="ruta/a/imagen.jpg" alt="Descripción de la imagen">

Listas

Las etiquetas <ul> (lista no ordenada) y <ol> (lista ordenada) contienen <li> (elementos de lista):

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>

<ol>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ol>

Divisiones y Span

<div> define una sección en bloque, mientras que <span> define una en línea:

 <div>
     <h2>Encabezado en div</h2>
     <p>Párrafo en div</p>
 </div>

 <p>Texto con <span style="color:red;">span rojo</span& incluido.</p>

Formularios

La etiqueta <form> define un formulario con campos <input> y etiquetas <label>: 

<form action="/enviar" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>

Tablas

La etiqueta <table> define una tabla con filas (<tr>) y celdas (<td>, <th>):

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
    </table>

Etiquetas Semánticas

Estas etiquetas añaden significado al contenido:

  • <header>: Encabezado de la página o sección.
  • <nav>: Sección de navegación.
  • <section>: Sección de un documento.
  • <article>: Contenido independiente.
  • <aside>: Contenido adicional.
  • <footer>: Pie de página.
<header>
    <h1>Mi Sitio Web</h1>
</header>
    
<nav>
    <ul>
        <li><a href="inicio">Inicio</a></li>
        <li><a href="acerca">Acerca de</a></li>
        <li><a href="contacto">Contacto</a></li>
    </ul>
</nav>
    
<section>
    <h2>Sección Principal</h2>
    <p>Contenido de la sección principal.</p>
</section>
    
<aside>
<h3>Información Adicional</h3>
<p>Contenido adicional.</p>
   </aside>
    
<footer>
    <p>© 2024 Mi Sitio Web</p>
</footer>

Buenas Prácticas

  • Usa etiquetas semánticas para mejorar la accesibilidad y el SEO.
  • Cierra todas las etiquetas, incluso las que no lo requieren explícitamente, como <img>.
  • Escribe un código limpio y ordenado.
  • Documenta el código con comentarios.
  • Valida tu HTML con herramientas como W3C Markup Validation Service.
