Teoría de CSS Grid


/* CSS Grid: Definición y Ejemplo */
CSS Grid Layout es un sistema de diseño bidimensional que permite crear layouts complejos con filas y columnas. Utiliza una cuadrícula para organizar el contenido de una página.

Ejemplo de CSS Grid:

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* Tres columnas de igual ancho */
    grid-template-rows: auto; /* Fila automática */
    gap: 10px; /* Espacio entre elementos */
}

.item {
    background-color: #e0e0e0;
    padding: 20px;
    text-align: center;
}

Teoría del Versionado de Código



El versionado de código es una práctica que permite llevar un registro de los cambios realizados en el código fuente a lo largo del tiempo. Esto es útil para revertir a versiones anteriores en caso de errores y para colaborar en equipos.

Ejemplo de uso de Git para versionado:

1. Inicializar un repositorio:
   <code>git init</code>

2. Agregar archivos al índice:
   <code>git add .</code>

3. Realizar un commit:
   <code>git commit -m "Descripción del cambio"</code>

4. Ver el historial de commits:
   <code>git log</code>
